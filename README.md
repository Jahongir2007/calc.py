### Calc.py easy python library for calculate
You will be able to add, subtract, multiply and divide two numbers through this library.
```markdown
import calcpy
cpy = calcpy

#Add numbers
cpy.add(12, 5)

#Subtract numbers
cpy.multipi(14, 7)

#Multiply numbers
cpy.multipl(12, 5)

#Divide numbers
cpy.div(12, 6)
```
